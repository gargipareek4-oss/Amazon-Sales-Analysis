# Amazon Sales Analysis

## Project Overview

This project analyzes Amazon sales data to identify sales trends, top-performing product categories, regional performance, and fulfillment efficiency.

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- GitHub

## Project Objectives

- Clean and preprocess sales data
- Analyze sales performance
- Identify top-selling categories
- Analyze state-wise and city-wise sales
- Study monthly sales trends
- Generate business insights

## Dataset Information

- Total Records: ~129,000
- Features: 21 Columns
- Data Source: Amazon Sales Dataset

## Analysis Performed

### Data Cleaning
- Removed null values
- Removed unnecessary columns
- Converted date columns

### Exploratory Data Analysis
- Total Sales
- Total Orders
- Average Order Value
- Top Categories
- Top States
- Top Cities
- Fulfillment Analysis
- Monthly Sales Trend

## Key Insights

1. Top product categories generated the highest revenue.
2. Maharashtra and Karnataka were among the highest revenue-generating states.
3. Metropolitan cities contributed significantly to sales.
4. Monthly sales trends showed variation in customer demand.
5. Fulfillment methods impacted sales distribution.
